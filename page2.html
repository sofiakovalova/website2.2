HTML — мова розмітки гіпертексту

HTML (HyperText Markup Language) — це основа будь-якої веб-сторінки. Вона описує структуру документа, визначає абзаци, заголовки, посилання, зображення, таблиці та інші елементи.

Наприклад, щоб створити абзац, використовують тег:

<p>Це абзац тексту.</p>

Сучасний стандарт HTML5 включає семантичні теги, мультимедіа та API для взаємодії з браузером.

HTML — це кістяк будь-якої веб-сторінки. Пам’ятай: контент завжди відділяй від стилів.
